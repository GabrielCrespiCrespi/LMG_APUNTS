Estructura bàsica

Un document HTML es basa en una estructura jeràrquica de etiquetes que defineixen les diferents parts d'una pàgina, Cada pàgina HTML segueix una estructura similar, començant pel <!DOCTYPE>, seguit de les etiquetes <html> , <head>, <body>.

Etiquetes bàsiques

    <title> - serveix per definir el títol que apareix a la pestanya del navegador o als resultats dels cercadors.

Elements

    <i> i <em> - S'utilitza per posar el text en cursiva
