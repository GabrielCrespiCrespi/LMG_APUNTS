Colors i fons

El CSS proporciona diverses propietats per aplicar colors i fons als elements HTML. Aquestes propietats són essencials per millorar l'aspecte visual i la llegibilitat de la pàgina web.

  1. Propietat color: La propietat color s'utilitza per canviar el color del text dins d'un element. Pots definir els colors de diverses maneres:

    1. Noms de colors: utilitzar noms predefinits com red, blue, green, etc.
    2. Colors hexadecimals: utilitzar un codi hexadecimal que comença amb # seguit de sis caràcters (exemple: #FF0000 per a vermell).
    3. RGB: utilitzar valors de rgb() on cada component (vermell, verd, blau) s'especifica entre 0 i 255.
    4. RGBA: igual que rgb(), però amb un quart valor per a l'opacitat (valors entre 0.0 i 1.0).

  2. Propietat background-color: La propietat background-color s'utilitza per establir el color de fons d'un element. Igual que la propietat color, pots definir colors utilitzant noms, valors hexadecimals, RGB o RGBA.

  3. Propietat background: La propietat background és una manera abreujada d'especificar diverses propietats de fons alhora, com el color, la imatge de fons, la repetició, la posició, etc.

  4. Propietat background-image; Aquesta propietat permet afegir una imatge com a fons d'un element. Pots especificar la ubicació de la imatge amb una URL. Si vols que la imatge es repeteixi o es mostri d'una altra manera, pots utilitzar altres propietats complementàries:

    1. background-repeat: Controla si la imatge de fons es repeteix. Els valors poden ser repeat, no-repeat, repeat-x (horitzontalment) o repeat-y (verticalment).
    2. background-position: Defineix la posició de la imatge de fons (per exemple, center, top, left).
    3. background-size: Permet ajustar la mida de la imatge (per exemple, cover o contain per adaptar-se a la mida de l'element).

de les propietats posibles:

  1. color: Defineix el color del text.
  2. background-color: Estableix el color de fons d'un element.
  3. background-image: Afegeix una imatge de fons a un element.
  4. background-repeat: Controla la repetició de la imatge de fons.
  5. background-position: Defineix la posició de la imatge de fons.
  6. background-size: Ajusta la mida de la imatge de fons.
  7. background: Propietat abreujada que combina diverses propietats de fons.
