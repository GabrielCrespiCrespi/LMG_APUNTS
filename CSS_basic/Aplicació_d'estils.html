Aplicació d'estils

El CSS es pot aplicar a un document HTML de tres maneres principals: en línia, incrustat (dins de l'HTML) i extern (mitjançant un fitxer CSS separat). Cadascuna d'aquestes formes té els seus propòsits i usos.

  1. Estils en línia

    2. Els estils en línia s'apliquen directament a un element HTML, utilitzant l'atribut style dins de la mateixa etiqueta de l'element. Aquest mètode és adequat per a aplicar estils a un sol element de manera ràpida, però no és recomanable per a pàgines grans, ja que dificulta la mantenibilitat.

    1. Aventatges: Aplicació ràpida i directa a l'element.
    2. 1. Desventatges: Dificulta la mantenibilitat en llocs webs grans perquè caldria aplicar l'estil a cada element manualment.
      2. No aprofita la capacitat de reutilitzar els estils.

  2. Estils incrustats

    3. Els estils incrustats es col·loquen dins de la secció <head> d'un document HTML, dins d'una etiqueta <style>. Aquests estils s'apliquen a tots els elements HTML de la pàgina actual.

    1. Aventatges: Més eficient que els estils en línia, ja que es poden aplicar estils a diversos elements d'una mateixa pàgina des d'un sol lloc.
    2. 1. Desventatges: Dificulta la mantenibilitat i llegibilitat en llocs webs grans perquè caldria aplicar l'estil a cada element manualment.
      2. Només afecta la pàgina on s'aplica, no es reutilitza entre diverses pàgines.

  3. Estils externs

    4. Els estils externs s'emmagatzemen en un fitxer CSS separat, i s'enllacen al document HTML utilitzant l'etiqueta <link> dins de la secció <head>. Aquest és el mètode més recomanat per a la majoria de llocs webs, especialment si consten de múltiples pàgines.

    1. Aventatges: 1.Permet reutilitzar els estils en diverses pàgines web, facilitant la consistència i mantenibilitat.
      2.Facilita la separació entre el contingut (HTML) i l'estil (CSS).
      3. Optimitza el temps de càrrega del lloc web, ja que els navegadors poden emmagatzemar en memòria cau els fitxers CSS.
    2. Desventatges: Si el fitxer CSS extern no es carrega correctament, la pàgina es mostrarà sense estils.
